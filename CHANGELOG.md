# Changelog

## 0.2.0

- Added `autore watch` for real-time run inspection
- Added per-iteration Codex stdout/stderr log streaming to files
- Added configurable timeouts for Codex, verify, and guard commands

- Added a reproducible `examples/demo-repo/` walkthrough

- Initial Codex-native autoresearch runner
- `init`, `run`, `status`, `doctor`
